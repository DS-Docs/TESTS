1. What is the output of the following code?

else:

2. Which loop is guaranteed to execute at least once?

A) for loop

B) while loop

C) do-while loop

D) Python doesn't have such a loop

Answer: D) Python doesn't have such a loop

Explanation: Python doesn't have a built-in do-while loop like some other languages. Both for and while loops may execute zero times if their conditions aren't met initially.

3. What does the 'break' statement do in Python?

Explanation: The break statement is used to exit a loop prematurely, regardless of the loop's condition.
