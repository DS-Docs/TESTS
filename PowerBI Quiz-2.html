1. What is the primary purpose of Power BI?

A) To create and share interactive data visualizations
B) To write complex programming algorithms
C) To design websites
D) To manage databases

Answer: A) To create and share interactive data visualizations

Explanation: Power BI is a business analytics service by Microsoft that provides interactive visualizations with self-service business intelligence capabilities, where end users can create reports and dashboards by themselves without having to depend on information technology staff or database administrators.

2. Which of the following is NOT a component of Power BI?

A) Power BI Desktop
B) Power BI Service
C) Power BI Mobile
D) Power BI Terminal

Answer: D) Power BI Terminal

Explanation: The main components of Power BI are Power BI Desktop (for report creation), Power BI Service (cloud-based for sharing and collaboration), and Power BI Mobile (for viewing reports on mobile devices). There is no component called "Power BI Terminal."

3. What is DAX in Power BI?

A) A data visualization technique
B) A formula expression language
C) A type of chart
D) A data source connector

Answer: B) A formula expression language

Explanation: DAX (Data Analysis Expressions) is a formula language used in Power BI, Power Pivot, and SSAS Tabular models. It's used to create custom calculations and aggregations in your data model.

4. Which Power BI feature allows you to ask questions about your data in natural language?

A) Quick Insights
B) Q&A
C) Power Query
D) DAX Studio

Answer: B) Q&A

Explanation: The Q&A feature in Power BI allows users to ask questions about their data using natural language and get answers in the form of visualizations. For example, you can type "show sales by region as a bar chart" and Power BI will create that visualization.

5. What is the purpose of Power Query in Power BI?

A) To create visualizations
B) To connect to and transform data
C) To write DAX formulas
D) To publish reports

Answer: B) To connect to and transform data

Explanation: Power Query is the data connection and transformation component of Power BI. It allows you to connect to various data sources, clean and transform the data, and load it into the Power BI data model.

6. Which of the following file formats can Power BI Desktop directly import?

A) .xlsx
B) .csv
C) .json
D) All of the above

Answer: D) All of the above

Explanation: Power BI Desktop can directly import all these file formats: Excel files (.xlsx), comma-separated values files (.csv), and JavaScript Object Notation files (.json), along with many other data source types.

7. What is a measure in Power BI?

A) A visual element on a report
B) A dynamic calculation based on user interaction
C) A static column in a table
D) A type of data connection

Answer: B) A dynamic calculation based on user interaction

Explanation: A measure is a DAX formula that calculates a value dynamically based on the current context (like filters or slicers applied by the user). Measures are recalculated as users interact with reports.

8. Which of these is NOT a valid Power BI visualization type?

A) Matrix
B) Gauge
C) Scatter chart
D) Hexagon map

Answer: D) Hexagon map

Explanation: While Power BI offers many visualization types including Matrix, Gauge, and Scatter chart, there is no built-in "Hexagon map" visualization. There are standard map visualizations and possibly custom visuals that might resemble this.

9. What is the purpose of bookmarks in Power BI?

A) To save specific views of a report
B) To mark important data points
C) To create relationships between tables
D) To optimize query performance

Answer: A) To save specific views of a report

Explanation: Bookmarks in Power BI capture the current state of a report page, including filters, slicers, and the visibility of objects. You can use bookmarks to create guided narratives or interactive report experiences.

10. Which DAX function calculates the sum of values in a column?

A) SUM()
B) TOTAL()
C) ADD()
D) CALCULATE()

Answer: A) SUM()

Explanation: The SUM() function in DAX adds all the numbers in a column. For example, SUM(Sales[Amount]) would calculate the total of all values in the Amount column of the Sales table.

11. What is the difference between a calculated column and a measure in Power BI?

A) Calculated columns are static, measures are dynamic
B) Measures are static, calculated columns are dynamic
C) There is no difference
D) Calculated columns can't use DAX

Answer: A) Calculated columns are static, measures are dynamic

Explanation: Calculated columns are computed during data refresh and stored in the model, while measures are calculated at query time based on the current context. Measures respond to user interactions like filters and slicers.

12. Which Power BI feature allows you to create a single visual that updates based on a selected field?

A) Hierarchies
B) Drillthrough
C) Field parameters
D) Conditional formatting

Answer: C) Field parameters

Explanation: Field parameters allow you to create dynamic visuals where the fields displayed can be changed by the report user through a slicer. This creates more interactive and flexible reports.

13. What is the purpose of the "Analyze in Excel" feature in Power BI?

A) To export Power BI data to Excel
B) To connect Excel to a Power BI dataset for pivot table analysis
C) To replace Power BI with Excel
D) To import Excel formulas into Power BI

Answer: B) To connect Excel to a Power BI dataset for pivot table analysis

Explanation: "Analyze in Excel" allows Excel users to connect to a Power BI dataset and create PivotTables, PivotCharts, and Excel formulas using the data model in Power BI, without needing to export data.

14. Which of the following is NOT a valid data modeling relationship in Power BI?

A) One-to-one
B) One-to-many
C) Many-to-many
D) All-to-all

Answer: D) All-to-all

Explanation: Power BI supports one-to-one, one-to-many, and many-to-many relationships, but there is no such thing as an "all-to-all" relationship in data modeling.

15. What does the "Performance Analyzer" in Power BI help you do?

A) Identify slow-performing visuals in a report
B) Analyze data quality
C) Optimize DAX formulas
D) Clean data sources

Answer: A) Identify slow-performing visuals in a report

Explanation: The Performance Analyzer in Power BI Desktop records and displays the time taken to perform different operations (like visual rendering or DAX queries) helping you identify and troubleshoot performance bottlenecks.

16. Which DAX function is used to create a filter context in a calculation?

A) FILTER()
B) CALCULATE()
C) SUMX()
D) RELATED()

Answer: B) CALCULATE()

Explanation: CALCULATE() is the most important function in DAX as it allows you to modify the filter context of a calculation. It evaluates an expression in a modified filter context.

17. What is the purpose of the "Drillthrough" feature in Power BI?

A) To navigate between report pages
B) To see detailed data for a specific data point
C) To export data to Excel
D) To create hierarchies

Answer: B) To see detailed data for a specific data point

Explanation: Drillthrough allows users to right-click on a data point and "drill through" to a dedicated page that shows details about that specific item, with the context passed from the original selection.

18. Which Power BI feature allows you to create a custom navigation experience in your reports?

A) Buttons
B) Bookmarks
C) Selection pane
D) All of the above

Answer: D) All of the above

Explanation: You can create sophisticated navigation experiences in Power BI using a combination of buttons (to trigger actions), bookmarks (to save specific views), and the selection pane (to control visibility of objects).

19. What is the purpose of the "Enter Data" feature in Power BI Desktop?

A) To manually input small datasets
B) To edit existing data tables
C) To create calculated columns
D) To import data from APIs

Answer: A) To manually input small datasets

Explanation: The "Enter Data" feature allows you to manually create a new table by typing or pasting data directly into Power BI Desktop, which is useful for small reference tables or configuration data.

20. Which of these is NOT a valid data transformation in Power Query?

A) Pivot columns
B) Unpivot columns
C) Transpose table
D) Rotate table 45 degrees

Answer: D) Rotate table 45 degrees

Explanation: While Power Query can pivot, unpivot, and transpose tables (switch rows and columns), it cannot rotate a table at an angle like 45 degrees - that would be a visual transformation, not a data transformation.

21. What is the difference between "Import" and "DirectQuery" storage modes in Power BI?

A) Import loads data into Power BI, DirectQuery queries the source directly
B) Import is faster but uses more memory
C) DirectQuery shows real-time data but may be slower
D) All of the above

Answer: D) All of the above

Explanation: All statements are correct. Import mode loads data into Power BI's in-memory engine (fast but requires refresh). DirectQuery leaves data in the source and queries it live (real-time but potentially slower).

22. Which DAX function would you use to find the average sales for the last 3 months?

A) LASTDATE()
B) DATESINPERIOD() with AVERAGE()
C) SUM()/COUNT()
D) TOPN()

Answer: B) DATESINPERIOD() with AVERAGE()

Explanation: You would use a time intelligence function like DATESINPERIOD() to define the 3-month period, combined with AVERAGE(). For example: AVERAGEX(DATESINPERIOD(Sales[Date], LASTDATE(Sales[Date]), -3, MONTH), Sales[Amount]).

23. What is the purpose of the "Selection Pane" in Power BI Desktop?

A) To select data sources
B) To manage the visibility and layering of visuals on a page
C) To choose which columns to import
D) To select DAX functions

Answer: B) To manage the visibility and layering of visuals on a page

Explanation: The Selection Pane shows all objects on the current report page and allows you to control their visibility (eye icon) and z-order (up/down arrows), which is especially useful when working with bookmarks.

24. Which Power BI feature allows you to define how visuals on a page respond to different screen sizes?

A) Themes
B) Responsiveness settings
C) View mode
D) Mobile layout

Answer: B) Responsiveness settings

Explanation: The responsiveness settings (available in the Visualizations pane) allow you to define how visuals should behave when the report page is resized, including options to move, resize, or hide visuals based on available space.

25. What is the purpose of the "Field Parameters" feature in Power BI?

A) To create dynamic measures
B) To make visuals interactive based on user selection
C) To optimize query performance
D) To define relationships between tables

Answer: B) To make visuals interactive based on user selection

Explanation: Field parameters allow you to create dynamic reports where users can select which fields to display in visuals through slicers. This creates more flexible and interactive reports without needing to create multiple similar visuals.

26. Which of the following is NOT a valid way to share a Power BI report?

A) Publish to Power BI service
B) Share via Microsoft Teams
C) Export to PowerPoint
D) Save as executable (.exe) file

Answer: D) Save as executable (.exe) file

Explanation: Power BI cannot export reports as executable files. You can publish to the Power BI service, share via Teams, or export to PowerPoint (as static slides), among other sharing options.

27. What is the purpose of the "Performance Analyzer" in Power BI?

A) To analyze data distribution
B) To identify slow-performing visuals and queries
C) To optimize DAX formulas
D) To clean data sources

Answer: B) To identify slow-performing visuals and queries

Explanation: The Performance Analyzer records and displays detailed timing information about operations in your report, helping you identify which visuals or queries are causing performance issues.

28. Which DAX function would you use to calculate year-to-date sales?

A) TOTALYTD()
B) YEAR()
C) SUM()
D) CALENDAR()

Answer: A) TOTALYTD()

Explanation: TOTALYTD() is a time intelligence function that calculates the year-to-date total of an expression. For example: TOTALYTD(SUM(Sales[Amount]), 'Date'[Date]).

29. What is the purpose of the "Decomposition Tree" visual in Power BI?

A) To show hierarchical data
B) To analyze the drivers behind a metric
C) To display geographic data
D) To create organizational charts

Answer: B) To analyze the drivers behind a metric

Explanation: The Decomposition Tree is an AI-powered visual that allows users to explore data across multiple dimensions to understand what factors are contributing to a specific metric value.

30. Which Power BI feature allows you to create custom visuals beyond the built-in options?

A) Power BI Custom Visuals SDK
B) Power Apps
C) Power Automate
D) Power Query Editor

Answer: A) Power BI Custom Visuals SDK

Explanation: The Power BI Custom Visuals SDK (Software Development Kit) allows developers to create custom visualizations using TypeScript and D3.js, which can then be imported into Power BI reports.
